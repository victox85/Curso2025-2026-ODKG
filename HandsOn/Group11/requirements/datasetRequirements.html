Bus App Dataset Requirement Satisfaction

Overview

This dataset contains information regarding the schedules, stops, frequencies and other information of the Municipal Transport Company (EMT) of the city of Madrid.

Meeting the specified requirements:

R1. Data are in one of the smart city domains.

Yes. It is part of the city's effort to track the public transport to better inform the users, either through third party applications or official outlets that would have easier access to such information.

R2. Data are available as a CSV file.

Yes. All files are CSVs even if the extension is ".txt".

R3. Data have an open license so they can be published.

Yes. Its license allows for distribution of this information either through the same license or a different one.

R4. Data can be easily linked with generic real-world entities (e.g., locations).

Yes. The "stops" file for example, contains coordinates to be able to display them in a map.

R5. Documentation exists for the data (optional).

Yes. It can be found here. It relates more to Google Transit than solely on the dataset itself but it can be very useful.

R6. Data can come from multiple data sources (optional).

Data in this dataset only comes from the EMT. But, we think it has a high potential for integration with other datasets.
