Application Requirements

Purpose

Our application will help users find where to recharge their electrical vehicules near them. We will have a map with the location marked, with also the number of people currently charging to fludify the circulation. Ideally we would also like to make the price users will have to pay.

Functional Requirements

  • The application must allow users to search for nearby charging stations.
  • The application must display the location of charging stations on a map.
  • The application must provide real-time information on the availability of charging stations.
  • The application must allow users to filter charging stations by type (e.g., fast charger, standard charger).
  • The application must provide directions to the selected charging station.
  • The application must allow users to rate and review charging stations.
  • The application must provide information on the pricing of charging stations.
  • The application must allow users to create an account and save their favorite charging stations.

Please note that this is what we would like to do, but it will depends on the available data and resources.

User interface

  1. Search Bar
    • Users can enter their location or allow the app to access their current location.
    • Users can enter specific criteria to filter results.
  2. Map View
    • Interactive map displaying the location of charging stations.
    • Each pin displays :
      • Name of the charging station
      • Type of charger
      • Availability status or Number of people currently charging
      • User ratings and reviews
      • Price information
  3. Station List
    • Lists of nearby station with :
      • Address
      • Name of the charging station
      • Type of charger
      • Availability status or Number of people currently charging
      • User ratings and reviews
      • Price information
