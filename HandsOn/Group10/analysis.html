Analyse Licensing of the Data Set

• Who is the publisher and the rightsholder?

In this cases, the publisher and the rightsholder is the same entity: the **Townhall of Barcelona**.

• What is the license?

The license is **Creative Commons 4.0** for all the data selected.

• Which will be the license to be used for the generated dataset?

We will use the same license **Creative Commons 4.0**.

https://creativecommons.org/licenses/by/4.0/deed.es
