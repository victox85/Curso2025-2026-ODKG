1) Educational centers in the Community of Madrid

Catalog page link: datos.gob.es — Educational centers

  • R1 — Smart city: essential public service (education) and urban planning; the description explicitly includes identification and location data for the centers.
  • R2 — CSV: a CSV distribution is available with a direct URL to the resource centros_educativos.csv.
  • R3 — License: Creative Commons BY 4.0 (attribution required).
  • R4 — Linkability: the CSV contains identifying and location fields (e.g., address, municipality, postal code); these allow linking with territorial bases (e.g., barrios/districts, geocoding, OSM).
  • R5 — Documentation: the page provides complete metadata (distributions, format, size, identifiers, update date).
  • R6 — Multi-source (opt.): no multi-source integration stated (likely a single origin from the Community of Madrid).

2) Traffic accidents in the city of Madrid

Catalog page link: datos.madrid.es — Accidents

  • R1 — Smart city: Transport sector, urban accident data recorded by the General Directorate of the Municipal Police.
  • R2 — CSV: CSV downloads are available for each year (2010–2025) in the Downloads section.
  • R3 — License: link to the portal’s Terms of use (reuse permitted with source citation).
  • R4 — Linkability: location fields (street/number, district) and UTM coordinates (to be converted to WGS84) enable spatial joins with schools and speed cameras.
  • R5 — Documentation: the PDFs “Estructura del fichero de datos de accidentes de tráfico” for 2019–… and 2010–2018 are available, including a data dictionary.
  • R6 — Multi-source (opt.): no multi-source integration indicated; single operational source (Municipal Police).

3) Fixed speed cameras — Madrid

Catalog page link: datos.madrid.es — Speed cameras

  • R1 — Smart city: Transport sector; georeferenced information on fixed speed cameras.
  • R2 — CSV: Downloads section with CSV files (also XLSX and GEO).
  • R3 — License: link to the portal’s Terms of use.
  • R4 — Linkability: longitud/latitud fields in WGS84 (World Geodetic System 84) ready for spatial joins.
  • R5 — Documentation: description and basic metadata; page with map visualization.
  • R6 — Multi-source (opt.): no multi-source integration indicated; two responsible departments but an internal administrative source.
